    Features of the app includes:

    1. Emergency request and response – connect with the designated responders depending on the emergency type and available emergency coverage. the sole purpose of the user location data is to specifically know the user location for emergency response purposes the app gets the user location and it would thene be passed to the emergency responders for navigation to user emergency location. user app retrieves user metadata such as location data (latitude, longitude) and sent to dispatcher via API to save to database. dispatcher will then fetch the information and send the emergency details including collected location data (latitude, longitude) to the responder app. the location data (latitude, longitude) will be used for locating of the emergency. Authenticated users by means of the emergency request function module allows the user to send distress signal to the authorities and available responders. This uses the mobile geo-location services along with the user information from KYC to specifically identify user location details that would be sent out to the command center on the Dispatch admin portal to assess the situation. The Dispatch Admin Portal would then need to deploy and assign applicable emergency responders to the user location. User data including the KYC information and the user geo-location will be used by the emergency responders to navigate through the shortest path to attend to emergency distress signal.
    2. News and Updates - App as a medium to connect with the local authorities on the announcement and activities. This is only being broadcasted by the verified local government authorities. As such, this also eliminates fake news and misinformation within the community.
    3. Community Connect – connecting people and local small businesses to promote collaboration and support. By means of free advertising, users with small businesses can post their products and services for free and be available to users within the area. This allows the community to support and prosper entrepreneurs as well as help community users to avail legitimate and available nearby products and services.
    4. Health – avenue for FAQs and how To’s related to health, vaccines and announcements. This is coming from the local healthcare professionals to provide insightful health announcements and provide meaningful and healthier society.
    5. Social Services – connecting people with local authorities on barangay IDs and other related documents. Users can notify the local authorities on their document requirement. Users can fill up the available forms that would then be sent out to the HQ admin portals that would notify the local authorities on the document request. Manual processing and BAU activities will be handled separately by the administrators and document processors.
